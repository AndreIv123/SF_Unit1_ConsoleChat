## Список участников

Проект выполнен самостоятельно.

## Задание

Создайте чат со следующим функционалом:

- консольная программа

- регистрация пользователей - логин, пароль, имя

- вход в чат по логину/паролю

- отправка сообщений конкретному пользователю

- обмен сообщениями между всеми пользователями чата одновременно


## Описание решения

Программа консольная, для навигации используется текстовое меню.
Чат представлен массивом (указателей) пользователей.
Каждый пользователь хранит разделяемые указатели на адресованные ему сообщения.
Сообщение удаляется после прочтения всеми адресатами.
